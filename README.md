# Advo AI - O'zbekiston uchun huquqiy yordamchi

⚖️ **Advo AI** - O'zbekistonning eng ilg'or raqamli huquqiy advokati.

## Xususiyatlar

### 1. Huquqiy Vaziyat Tahlili
- Mehnat, fuqarolik, oilaviy masalalarni tahlil qilish
- O'zbekiston qonunlariga aniq havolalar
- Amaliy harakat rejasi

### 2. Shartnoma Tekshiruvi ("Blind Sign" Protector)
- Foydalanish shartlari va Maxfiylik siyosatini tahlil qilish
- Yashirin tuzoqlarni aniqlash
- Xavf darajasini baholash (0-100 ball)

## O'rnatish

```bash
# Dependencies o'rnatish
npm install

# .env.local faylini yarating va GEMINI_API_KEY qo'shing
# API kalitini olish: https://aistudio.google.com/app/apikey

# Dev serverni ishga tushirish
npm run dev
```

## Muhit o'zgaruvchilari

`.env.local` faylini yarating:

```
GEMINI_API_KEY=your_gemini_api_key_here
```
